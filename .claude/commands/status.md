# /status - 進捗確認コマンド

現在のプロジェクト進捗状況を確認し、次に取り組むべきタスクを提案します。

## 実行内容

1. **`docs/plan.md` の完了率を計算**
   - Phase 1-4ごとの進捗率を表示
   - 完了済み（`[x]`）と未完了（`[ ]`）のタスク数を集計

2. **`docs/session-log.md` の最新エントリを表示**
   - 最後に完了したタスク
   - 次回の作業予定
   - 重要な備考・ブロッカー

3. **次のタスクを提案**
   - 優先度に基づいた次のアクション
   - 関連ファイル・ドキュメントへのリンク
