# コマンドリファレンス

## 概要

Tennis Discovery AgentのDiscordコマンド一覧です。

---

## 基本的な使い方

### チャンネルベースの操作（メイン）

コマンドではなく、チャンネルを選択して音声/テキストを送信します。

| チャンネル | 操作 | 説明 |
|-----------|------|------|
| #壁打ち | 音声/テキスト送信 | 壁打ち練習の記録 |
| #スクール | 音声/テキスト送信 | レッスンの記録 |
| #試合 | 音声/テキスト送信 | 試合の記録 |
| #フリー練習 | 音声/テキスト送信 | 自由練習の記録 |
| #振り返り | 音声/テキスト送信 | 過去メモへの追記 |
| #質問 | 音声/テキスト送信 | 過去メモを参照した質問（Phase 2） |
| #分析 | 音声/テキスト送信 | 成長分析リクエスト（Phase 3） |

---

## 管理コマンド（Phase 2以降）

### /status

Botの状態を確認します。

```
/status
```

**出力例:**
```
🤖 Tennis Discovery Agent
状態: オンライン
保存したメモ: 127件
最終更新: 2025-01-27 14:30
```

### /search（Phase 3）

キーワードでメモを検索します。

```
/search キーワード
```

**使用例:**
```
/search バックハンド
```

---

## ボタン操作（Phase 2以降）

音声メモ送信後に表示されるオプションボタン：

| ボタン | 説明 |
|--------|------|
| 🤔 深堀質問 | AIが追加質問を生成 |
| 📊 過去と比較 | 過去のメモと比較 |
| 🔖 重要マーク | 重要なメモとしてマーク |
| ✅ 保存のみ | そのまま保存（デフォルト） |

---

## DM操作（Bot停止時）

Bot停止時の緊急記録方法：

1. 自分のDMにBot宛てで音声を送信
2. Bot復旧後に自動処理
3. 処理済みメッセージに✅がつく

**注意:** ADMIN_USER_IDの設定が必要
