## 第1章: はじめての記録（Week 1）

### 🎾 Day 1: 最初の壁打ち

田中さんは、テニス歴1年。最近スランプに陥り、サーブが安定しない。友人に勧められてTennis Discovery Agentを使い始めることにした。

**壁打ち練習後、Discordの`#壁打ち`チャンネルで音声メモ:**

> 「今日は30分壁打ちをしました。サーブが全然安定しなくて、特に2ndサーブがネットに引っかかることが多かったです。トスが低いのかな...？次は、トスの高さを意識してみようと思います。」

**Botの応答:**

```
✅ 壁打ち練習を記録しました！

📝 今回の練習
- 練習時間: 30分
- 課題: 2ndサーブがネットに引っかかる
- 仮説: トスが低い可能性
- 次回のテーマ: トスの高さを意識

📊 GitHubにプッシュ完了
🎯 Embedding生成完了
```

**田中さんの感想:** 「音声で話すだけで記録できるのは楽だな。次回のテーマも自動で設定してくれる。」

## 第2章: 気づきの深化（Week 2-4）

### 🎾 Week 2: ソクラテス式問答を体験

壁打ちで「良い感覚」を感じたが、うまく言語化できない。

**音声メモ:**

> 「今日は調子が良かった！サーブがスパッと決まる感覚があった。でも、何が良かったのかはっきりしない...」

**Botの応答（ボタン付き）:**

```
✅ 壁打ち練習を記録しました！

💭 良い感覚を感じたようですね。
もう少し詳しく振り返ってみませんか？

[🔍 もっと詳しく] [📊 過去と比較] [💾 記録を保存]
```

**田中さんが「🔍 もっと詳しく」をクリック:**

**Botの質問:**

```
🤔 その「スパッと決まる感覚」について教えてください。

- 身体のどの部分を意識していましたか？
- 打つ直前、どんな感覚がありましたか？
- 前回と何が違いましたか？
```

**田中さんが音声で回答:**

> 「そういえば、トスを上げる時に左手の力が完全に抜けていた気がします。前は、力を入れて高く上げようとしていたけど、今日は肩から自然に上げる感覚でした。」

**Botの応答:**

```
💡 素晴らしい発見です！

「左手の脱力」と「肩から自然に上げる」が
今日の好調の鍵だったんですね。

これは、スクールで学んだ
「トスは手首ではなく肩から」に繋がっています。

🎯 次回も、この感覚を意識してみましょう。

[💾 記録を保存]
```

**田中さんの感想:** 「ボットが質問してくれることで、自分でも気づかなかった『脱力』という感覚に気づけた。これ、大事だな。」
