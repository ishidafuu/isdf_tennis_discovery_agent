# 使い方ガイド

Tennis Discovery Agentを実際に使う際のガイドです。

## このセクションについて

**対象読者**: エンドユーザー（実際にシステムを使う方）

このセクションでは、Tennis Discovery Agentの日常的な使い方、具体的なユースケース、機能のリファレンスを提供します。

---

## 基本ガイド

### [基本的な使い方](./基本的な使い方.md)

日常的な使い方を詳しく解説したマニュアルです。

**含まれる内容:**
- Discord Botへのメモの送り方
- 各チャンネルの使い分け
- 音声メモの活用方法
- 画像・動画の添付方法
- 振り返りチャンネルでの過去ログ追記

**読むべき人:** 初めて使う方、基本操作を確認したい方

---

### [ストーリーガイド](./ストーリーガイド.md)

3ヶ月で上達するまでの具体的なストーリーです。

**含まれる内容:**
- テニススクール中級者Aさんの物語
- 月ごとの使い方の変化
- 実際の改善プロセス
- システムの価値を実感する場面

**読むべき人:** システムの全体像を掴みたい方、導入効果を知りたい方

---

### [機能一覧](./機能一覧.md)

Phase 1-4の全機能カタログです。

**含まれる内容:**
- Phase 1: 基盤機能（音声入力、構造化記録、サイクル管理）
- Phase 2: 対話機能（問いかけ、ボタンUI）
- Phase 3: データ活用（検索、統計、リマインド）
- Phase 4: 高度な分析（パターン検出、類似状況検索）

**読むべき人:** すべての機能を一覧で確認したい方
