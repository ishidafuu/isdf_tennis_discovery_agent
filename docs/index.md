## 推奨する読み順

### エンドユーザー向け

```
1. 使い方ガイド (ガイド/使い方ガイド.md)     # 基本的な使い方を理解
2. ストーリーガイド (ガイド/ストーリーガイド.md) # 実際の使用イメージを掴む
3. 機能一覧 (ガイド/機能一覧.md)            # すべての機能を確認
```

### 初回セットアップ時

```
1. クイックスタート.md                      # まず動かす
2. セットアップ/RaspberryPi.md              # 環境構築の詳細
3. セットアップ/Obsidian.md                # Obsidianの設定
```

### 実装開始時（開発者向け）

```
1. 概要/ビジョン.md                        # ビジョンを理解
2. 概要/アーキテクチャ.md                   # 全体像を把握
3. フェーズ/01-基盤強化/                   # Phase 1から順に実装
```

### 特定機能の実装時

```
技術/ から該当ドキュメントを参照
```
