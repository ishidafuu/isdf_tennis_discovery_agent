## 推奨する読み順

### エンドユーザー向け（実際に使う方）

```
1. 使い方/基本的な使い方.md              # 基本的な使い方を理解
2. 使い方/ストーリーガイド.md             # 実際の使用イメージを掴む
3. 使い方/機能一覧.md                    # すべての機能を確認
4. 使い方/リファレンス/                  # 必要に応じて参照
```

### 初回セットアップ時

```
1. セットアップ/クイックスタート.md       # まず動かす（開発環境）
2. セットアップ/dotenvxクイックスタート.md # 本番環境へのデプロイ
3. セットアップ/RaspberryPi.md           # Raspberry Pi環境構築の詳細
4. セットアップ/Obsidian.md             # Obsidianの設定
```

### 開発・実装時（開発者向け）

```
1. 開発/概要/ビジョン.md                # ビジョンを理解
2. 開発/概要/アーキテクチャ.md           # 全体像を把握
3. 開発/フェーズ/01-基盤強化/           # Phase 1から順に実装
4. 開発/技術/                          # 特定機能の実装時に参照
```
