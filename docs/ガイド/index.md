# ガイド

このセクションには、Tennis Discovery Agentの包括的なガイドドキュメントが含まれています。

## ドキュメント一覧

### [機能一覧](./機能一覧.md)
Phase 1-4の全機能を整理した機能カタログ。各機能の説明、実装ファイル、完了状況を確認できます。

### [アーキテクチャ](./アーキテクチャ.md)
システム全体の構成図、データフロー、モジュール依存関係、スケーラビリティ、セキュリティ層の詳細。

### [使い方ガイド](./使い方ガイド.md)
エンドユーザー向けの詳細な使い方ガイド。基本的な使い方からチャンネル別の使い方、応用機能、トラブルシューティングまで網羅。

### [ストーリーガイド](./ストーリーガイド.md)
架空のユーザー「田中さん」の3ヶ月間の上達ストーリー。全機能を自然な流れで紹介。

### [リファクタリング分析](./リファクタリング分析.md)
コードベース全体のリファクタリング箇所の洗い出し。重複コードの削減提案、型安全性の向上、推定工数を含む包括的な分析レポート。

---

## 推奨する読み順

### エンドユーザー向け
1. [使い方ガイド](./使い方ガイド.md) - 基本的な使い方を理解
2. [ストーリーガイド](./ストーリーガイド.md) - 実際の使用イメージを掴む
3. [機能一覧](./機能一覧.md) - すべての機能を確認

### 開発者向け
1. [アーキテクチャ](./アーキテクチャ.md) - システム全体像を把握
2. [機能一覧](./機能一覧.md) - 実装済み機能を確認
3. [リファクタリング分析](./リファクタリング分析.md) - 改善ポイントを理解
